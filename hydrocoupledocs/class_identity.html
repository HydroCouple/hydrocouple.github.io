QString Identity::id ( ) const
overridevirtual

Gets a unique identifier for the entity.

Returns the Id as a String. The Id must be unique within its context but does not need to be globally unique. E.g. the id of an input exchange item must be unique in the list of inputs of a IModelComponent, but a similar Id might be used by an exchange item of another IModelComponent.

Implements HydroCouple::IIdentity.
